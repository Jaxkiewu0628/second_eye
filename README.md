# Second Eye

帮助视障人士独立出行的 AI 眼镜。

**摄像头 + 端侧 YOLO 识别障碍物 → 骨传导语音告知方位。完全离线，断网不影响使用。**

---

## 产品定位

- **目标用户**：视障人士，出行需要帮助的盲人
- **核心功能**：实时识别台阶/行人/车辆/路缘，骨传导播报"前方 1.5 米台阶"
- **差异化**：端侧 AI 全离线 — 竞品（OrCam $4490、WeWalk $850）均无语义理解或云端依赖

# Second Eye — 当前状态

> 单一事实源。最后更新：2026-06-29

## 一句话现状
项目更名 **Second Eye**，方向从 AR 运动眼镜**全线转向视障辅助设备**。
用 RK3576 端侧 AI 技术栈，重新实现 2023 年 YCane 概念的升级版：
**摄像头 + YOLO 语义理解 + 骨传导语音**，替代当年的 ESP32 + ToF 简单测距。
硬件验证成果完整保留，切入点与痛点已明确（无需再找）。

---

## 核心定位与北极星

- **产品**：帮助视障人士独立出行的 AI 眼镜，**Second Eye**
- **核心功能**：实时识别前方障碍物（台阶/车辆/行人/门/路缘），通过骨传导告知方位与距离
  - 示例："前方 1.5 米有台阶，右侧 2 米有行人靠近"
- **差异化主线：端侧优先，离线全可用**
  - 竞品（OrCam $3500、WeWalk $500）无端侧 AI，断网/延迟=失效
  - 我们：摄像头→NPU→骨传导，全程本机，0 依赖网络
  - 这对视障用户是安全要求，不只是特性
- **已有 IP 保护**：实用新型专利已申请（申请号 2023207836095，2023-04-10）
- **切入点已定，无需再找**：1.73 亿视障人口，53% 出行需要帮助，痛点真实且已用户验证

## 技术升级对比（YCane 2023 → Second Eye 2026）

| 维度 | YCane 2023 (ESP32) | Second Eye 2026 (RK3576) |
|------|--------------------|--------------------------|
| 主控算力 | ESP32，无 AI | RK3576，6 TOPS NPU |
| 障碍感知 | ToF 点测距，只知道"有东西" | 摄像头 + YOLOv8，知道"是什么" |
| 反馈方式 | 蜂鸣/简单音调 | 骨传导语音："前方 1.5 米有台阶" |
| 网络依赖 | 无 AI，但功能也弱 | 核心功能完全离线 |
| 形态 | 手杖 1.36kg | 眼镜（轻量化） |
| 显示 | 无 | **无（不需要）**——音频反馈即可 |
| 探测角度 | 192.7°（竞品优势已有） | 摄像头广角 + ToF 精确测距融合 |

> ⚡ **关键简化**：视障用户不需要显示屏。去掉显示光机，硬件难度大幅降低，成本降低，重量减轻。这解锁了之前最大的技术瓶颈。

## 系统架构

```
摄像头(MIPI) ──→ ISP ──→ NPU: YOLOv8 障碍物检测
                              ├── 识别类别：台阶/车/人/门/路缘…
                              └── 输出：类别 + bbox + 估算距离

ToF LiDAR ────────────────→ 精确测距（cm 级）
                              └── 与 YOLO bbox 融合 → 精确定位

[融合结果] ──→ 逻辑层：优先级 + 过滤 + 语音合成
                              └── "前方 1.5 米有台阶"

骨传导耳机 ←── TTS（本地 Matcha+Vocos）←── 语音文本

麦克风 ──→ STT（本地 Paraformer）──→ 用户指令（"前面安全吗？"）
                              └── 本地推理 / 云端增强
```

## 硬件决策

### 验证板（已定，待采购）
- **野火鲁班猫3 LBC_3WB（4+32G，BTB）摄像头套餐**（含 IMX415 + 12V@2A）~¥600
- 用途：跑通 YOLO + STT + TTS 完整链路，演示用

### RK3576 的角色
- ⚠️ **不是量产镜框芯片**（功耗 1.2-2.8W，不合 <80g 眼镜），仅作验证平台
- ✅ 跑通软件、给投资人演示、或放进**分体计算盒**（视障辅助允许稍大体积）

### 量产主控候选
- **RV1126B ⭐**（3 TOPS，1mW AOV 待机，同 RKNN 生态，验证可迁移）— 倾向
- **爱芯 AX630C**（3.2 TOPS，<1.5W）— 备选
- **Unisoc W517**（2-4 TOPS，可穿戴生态）— 备选
- ❌ Snapdragon AR1（NDA，无融资拿不到）

### 新增传感器：ToF LiDAR
- 2023 版核心传感器，仍保留——摄像头做语义，ToF 做精确测距，互补
- 候选：VL53L5CX（8×8 阵列，ST，约 ¥30）或 OPT8241（TI）
- **待选型**（需查真机精度 + 接口兼容性）

## 板到货后验证四项功能
1. **YOLOv8 障碍物检测**：识别台阶/人/车，输出类别+距离
2. **STT → TTS 语音闭环**：本地"你说话→板子理解→语音回复"
3. **ToF + YOLO 融合**：距离精度与语义类别对齐
4. **连续运行功耗**：视障设备必须 always-on，测实际功耗是否可接受

## 待决策
- **ToF LiDAR 选型**：VL53L5CX vs OPT8241，需查接口 + RK3576 驱动支持
- **YOLO 模型微调**：通用 COCO 80 类 vs 视障场景专用数据集（台阶/盲道/斑马线/路缘）
- **持续运行散热**：视障设备不能占空比 sleep，需评估长时间满载温升
- **法规路径**：是否按医疗器械注册（影响认证周期和成本）
- **定价 vs 竞品**：OrCam $3500（摄像头读文字）、WeWalk $500（基础 ToF 手杖）

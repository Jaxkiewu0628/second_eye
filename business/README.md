# Second Eye — Business / CEO

The CEO / business side of **Second Eye**: affordable, offline, hands-free AI glasses that tell a blind person what's in their path. The realized successor to Jackie Wu's Conrad-winning "YCane."

> This folder is destined for the monorepo's `business/` directory (github.com/Jaxkiewu0628/second_eye). The `../engineering/` side is Jackie's and is READ-ONLY for us. Never store project docs in gstack/system storage.

- **`todo.md`** — action items and the validation gate.

## Current state (2026-06-30)
Path A: offline, sub-$700, no-subscription assistive glasses. **Wedge = mobility** (obstacle/path detection); reading/scene is a free bonus, never the pitch. Engineering has independently converged on the same direction (mobility-first, offline, + a ToF depth sensor, glasses + neck compute box). Next: outdoor obstacle demo + discovery calls.

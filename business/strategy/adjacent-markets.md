# 邻近市场探索 — 离线端侧感知能力能占领的其他蓝海 / Adjacent Markets

> **初步探索(头脑风暴,蓝海度待验证)**。2026-07-01。
> 起因:盲人市场是我们舒服的切入点,但很小(美国核心 ~1M)。问题:我们的核心能力(整合创新 + 数据飞轮 + **离线端侧算力**)在别的"需要离线算力"的领域,有没有**还没被发现并占领**的场景?
> **不动摇当前定位**:盲人 mobility 是 beachhead(V1 聚焦);以下是平台扩张想象 + 融资大故事的燃料,phase-2+。

## reframe:我们真正的可迁移能力
不是"盲人眼镜",是 **"在云到不了的地方,做便宜、实时、可靠的 AI 感知 + 反馈"**。
这个能力值钱,正因为有**四种"云不行"**的场景:
1. **延迟致命**(安全实时——避障、碰撞、危险预警)
2. **无网 / 弱网**(户外、工业现场、发展中地区)
3. **隐私敏感**(医疗、家庭、残障——数据不能上云)
4. **成本敏感**(海量便宜边缘设备,养不起云)

盲人避障 = 四条全占(延迟致命 + 无网也要用 + 隐私 + 便宜)。所以问:**还有哪些场景也踩这四条、且没人用"便宜+离线+可穿戴+数据飞轮"这套占领?**

## 邻近场景(诚实标注红海/蓝海 + 能力迁移度)
| 场景 | 为什么云不行 | 市场规模(粗估待核) | 能力迁移度 | 诚实蓝海判断 |
|---|---|---|---|---|
| **失智/认知障碍老人 离线可穿戴辅助**(防走失、环境提醒、简单导航) | 隐私(家属抗拒云监控)+ 安全(走失是急事)+ 便宜(海量) | 全球失智 ~5500万且快速增长,远大于盲人 | **高**(可穿戴+感知+骨传导提醒直接复用) | 🟡 老人科技很热,但多是 GPS/呼叫手表;"**端侧环境感知 + 认知辅助**"的组合有缝 |
| **高危作业工人 离线安全预警**(工地/工厂/矿:机械接近、危险区、碰撞) | 现场网络差 + 延迟不能容忍 + 合规 | B2B,付费能力强(安全预算) | **高**(避障→避机械/危险区,同能力) | 🟡 工业安全 AI 有人做(多是固定摄像头);"**便宜可穿戴 + 个人离线预警**"有缝 |
| **两轮车 离线碰撞/盲区预警**(电动车/摩托/自行车) | 碰撞毫秒级 + 便宜 + 无网 | 巨大(中国/东南亚/印度电动车海量) | **高**(避障→交通避障) | 🟡 汽车有 ADAS、两轮车相对少;便宜离线预警有缝,但中国竞争激烈 |
| **聋人 声音事件→触觉/视觉提醒**(门铃/警报/来车声) | 隐私 + 实时 + 离线 | 全球听障巨大 | 中(换成音频感知,同"端侧感知+替代反馈"框架) | 🟡 有一些 app,专用可穿戴少 |
| 消防/救援/军事 低能见度感知 | 无网 + 延迟致命 | 专业、高价 | 中 | 🔴 高端、有成熟玩家、门槛高,不适合初创起步 |

## 诚实提醒
- **"完全没被发现"的场景罕见**——上面每个都有人碰过。真正的空白是**特定组合的交叉点**(便宜 + 离线 + 可穿戴 + 安全级实时 + 数据飞轮),就像盲人避障那样——不是"这个领域没人",而是"这个领域没人用我们这套打法"。
- 要确认哪个**真蓝海**、哪个其实红海,得像上次竞品扫描一样**去核实**(谁在做、什么形态/价格、离线与否),不能拍脑袋。

## 战略含义(给融资故事)
- 这把公司从 **"一个小众盲人眼镜"** 抬到 **"离线端侧感知平台,盲人是第一个垂直"** ——同一套(整合 + 数据飞轮 + 端侧算力)复用到多个"云不行"的场景。
- **但纪律不变**:先在盲人 mobility 做深做透(beachhead),证明这套打法能赢,再谈迁移。邻近市场是**故事的天花板**,不是 V1 的分心项。

## CEO 定向(2026-07-01)
- 失智老人 **不做**(Frankie)。
- **高危工人 = 优先验证方向**(Frankie:听上去更大)。理由:B2B 付费强(企业为安全/合规买单),不像盲人要等报销;技术迁移度高。
- **关键提醒(诚实)**:高危工人是 **B2B**(卖企业),另一套认证 + 采购 + 渠道,**不是盲人的顺延**——是"平台第二个垂直"的想象,验证清楚再决定何时做;**不动摇盲人 beachhead**。

## ✅ 高危工人 — 验证结论(2026-07-01,`research/worker-safety-feasibility-2026-07.md`)
**判定:CONDITIONAL —— 现在别分兵;当"融资故事期权 + V2+ 迁移",不是第二产品线。**
- **有真缝(对抗式核实过)**:零基建 + 第一人称 + 端侧离线视觉 + 看**未打标签**危险 + 只警告佩戴者 + 个人价格 —— 无现成产品占据。所有 incumbent 都是三种之一:固定/车载摄像头、**只认已标记物**的傻标签、往主管仪表盘传的云端 puck。
- **几乎免费保留**:和盲人**同一引擎(~90% 复用)**,V1 向下 ToF 落差 = 工人"高处坠落"同一个原语。
- **但现在别做,三因**:① struck-by(被撞)我们**只能单向警告工人**,UWB 双向(工人+司机)= 结构性弱点;② 买家/GTM 完全不同(企业 EHS/保险,6–12 月采购,工伤 ROI)= **另一家公司的活**;③ 最痛垂直(矿山/油气)要**防爆认证**(ATEX/IECEx,没做过、致命门槛)。
- **最优用法**:守盲人核心;工人安全 = 融资大故事天花板("同一离线引擎,盲人是第一个垂直,工人安全是下一个 $50B 期权");**唯一动作 = 引擎做成"危险类无关"**(盲人路缘=工人边缘,几乎零成本保留期权)。
- **唯一可探细分(若探)**:**非防爆 户外孤立/巡回 现场&O&M**(电力线/水务/光伏风电 O&M/堆场/道路作业)—— 四条属性全占、无认证墙、无人触及;用保险/工伤 ROI + 隐私安全(离线不追踪)去 pitch。
- **硬 NO**:跌倒/IMU me-too、UWB 接近、certified machine-STOP、气体/孤立作业 puck、固定 CCTV CV、仓库固定叉车 CV、防爆核心。
- **市场(佐证故事,中等可信)**:工业可穿戴 ~$8B(2025)→~$51B(2036,~16% CAGR);可穿戴职安 ~$4B;需求真实(美国 2024 建筑死亡 1034、坠落 #1)。但我们能守的 slice 是窄的。

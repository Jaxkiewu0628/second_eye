# AR 运动眼镜外壳设计方案 —— 扁平刀锋镜腿(Blade Temple)

> 方案 A 落地版 · Oakley 运动墨镜风格 · 2026-06-28  
> 核心思路:铰链侧做成**扁平刀锋**,大平面兼作散热片,符合运动墨镜形象

---

## 一、设计概念

参考 Oakley 运动墨镜的刀锋镜腿:**镜腿不是圆柱,而是一片竖向高、横向薄的扁平刀片。**

这一个形状同时解决三件事:
1. **装得下计算板** —— 20×25mm 的 PCB 平铺进刀锋(竖着放),刀锋只需 ~6mm 厚
2. **散热** —— 扁平大平面(铝合金)就是天然散热片,面积大、贴外侧空气
3. **运动外形** —— 包裹式 + 刀锋腿 = 标准运动墨镜语言

> 关键:把厚度"摊平"成面积。圆鼓舱是往外鼓(丑、散热差),刀锋是往竖向展开(运动感、散热好)。

---

## 二、整机坐标系(统一术语,避免歧义)

每个部件用三个方向描述,全文一致:

```
   长度(L) = 沿镜腿前后方向(铰链→耳朵),总长约 145mm
   高度(H) = 竖向(上→下),即刀锋的"宽脸"
   厚度(T) = 横向(贴头←→外侧),刀锋的"薄边"
```

> 刀锋的特点:**H 大、T 小**。圆鼓舱是 H 小、T 大(往外凸)。

---

## 三、整机布局

```
            前框(包裹式单片镜)
   ┌──────────────────────────────────────────┐
   │   摄像头        鼻托        [显示模块]      │
   └──┬────────────────────────────────────┬──┘
    铰链L                                 铰链R
   ┌──┴─────────────┐            ┌─────────┴──────────┐
   │  左刀锋镜腿      │            │   右刀锋镜腿        │
   │  电池为主        │            │   计算岛 + 电池     │
   │  (与右对称)      │            │   (主散热侧)        │
   └─────────────────┘            └────────────────────┘
```

**左右对称做成同样的刀锋外形**(运动墨镜美学),内部分工不同:右侧装计算,左侧多装电池配重。外观看不出差异。

## 四、精确尺寸

### 4.1 右镜腿(计算侧)— 变截面刀锋

```
段位      铰链弯折   计算刀锋段        电池过渡段       耳钩末端段
─────────────────────────────────────────────────────────────────
长度 L    15mm       45mm             40mm            45mm
高度 H    20mm       26mm  ──渐变──→   18mm  ──渐变──→  12mm
厚度 T    5mm        6mm              5mm             5mm
内容      FPC过线    计算岛+散热       电池条           扬声器+充电口
```

**计算刀锋段内部叠层(厚度 T 方向,共 6mm):**
```
外侧铝合金壳(兼散热)  1.0mm
  导热垫              0.5mm
  RK3576 计算 PCB     3.0mm   ← 20×25mm 平铺,竖向占 H
  内侧间隙 + 内壳      1.5mm
  ─────────────────────────
  合计                6.0mm
```

**计算岛元件清单(平铺在 20×25mm PCB 上):**

| 器件 | 占板尺寸 | 高度 |
|------|---------|------|
| RK3576 (FCBGA) | ~15×15mm | 1.2mm |
| LPDDR5 | ~10×12mm | 1.0mm |
| eMMC 32GB | ~11×10mm | 1.0mm |
| PMIC + NDP120 + 被动件 | 散布 | ≤1mm |

> PCB 竖放:25mm 沿长度 L,20mm 沿高度 H → 所以计算段高度 H 需 ≥24mm,取 26mm。

### 4.2 左镜腿(电池侧)— 同形刀锋

```
段位      铰链弯折   主电池段          过渡段          耳钩末端段
─────────────────────────────────────────────────────────────────
长度 L    15mm       45mm             40mm            45mm
高度 H    20mm       26mm  ──渐变──→   18mm  ──渐变──→  12mm
厚度 T    5mm        5mm              5mm             5mm
内容      FPC过线    电池×2            音频小板         扬声器
```
> 外形与右侧完全对称;因无计算岛,厚度可少 1mm(5mm),内部全是电池,做配重平衡。

### 4.3 电池规格(贴合刀锋的扁平条)

| 位置 | 单体尺寸(L×H×T) | 容量 | 数量 |
|------|----------------|------|:---:|
| 右刀锋 | 30 × 18 × 3.5mm | ~100mAh | 1 |
| 左刀锋 | 35 × 20 × 3.5mm | ~120mAh | 2 |
| **合计** | | **~340mAh** | 3 |

> 占空比模型下(见 architecture_brief),340mAh 触发式使用可达 14h+,连续录像约 1.8h。

### 4.4 前框(包裹式)

| 器件 | 尺寸 | 位置 | 备注 |
|------|------|------|------|
| 包裹镜片 | 单片弧形 | 整个前框 | 运动墨镜风格,大视野 |
| 摄像头(IMX415) | 15×15mm 模组 | 前框中央偏右 | 镜头凸出 ≤3mm |
| 显示模块(待定) | 待选型 | 右镜片后方 | 波导/Micro-OLED |
| 麦克风×2 | 3×3mm | 前框两端 | MEMS,贴 FPC |
| 鼻托 | 可调硅胶 | 中央 | 运动防滑 |

**前框内腔深度(镜片→前框背):8-12mm**

## 五、散热路径(刀锋的核心优势)

```
RK3576 发热(峰值 600mW,占空比下大部分时间休眠)
   │ 导热垫 0.5mm
   ▼
计算刀锋外侧铝合金面(26mm H × 45mm L ≈ 12cm² 大平面)
   │ 自然对流(贴外侧空气,非贴头)
   ▼
环境空气 + 运动气流(运动时迎面风加速散热)
```

**刀锋比圆鼓舱散热好的原因:**
- 散热面积大 3-4 倍(扁平展开 vs 圆鼓)
- 发热面朝**外侧**(迎风),不贴太阳穴 → 不烫脸
- 运动时有气流,被动散热效率进一步提升

> 外壳表面温度设计上限 40°C(皮肤安全)。

## 七、保留给外观组沟通的问题

> 这些是设计岔路,需要外观组拍板,工程侧已给出约束。

1. **🔴 刀锋高度 26mm 可接受吗?**
   计算段必须 ≥24mm 高才能平铺 PCB。再矮就装不下 RK3576,只能往厚度方向鼓(变回圆舱)。
   → 问:运动墨镜刀锋腿一般多高?26mm 在不在合理审美范围?

2. **🔴 帽子兼容**
   高尔夫戴帽子,刀锋腿铰链段会不会蹭帽檐/帽带?需镜腿弧度避让。
   → 需要外观组拿真帽子比划。

3. **🟡 左右对称 vs 功能优先**
   左右都做刀锋更对称好看,但左侧其实只需放电池。是否为了轻量允许左侧略薄/略矮?

4. **🟡 包裹镜片与显示模块**
   包裹式单片镜怎么和右眼显示模块结合?显示区开窗还是镜片内嵌?等显示选型定。

5. **🟡 铰链强度**
   刀锋薄(5mm)+ FPC 过线,铰链既要转动又要走线,机械强度和弯折寿命(>5万次)如何保证?

6. **🟢 材质分工**
   计算侧刀锋用铝合金(散热),其余段可用塑料(减重)?还是整体铝合金(质感统一但偏重)?

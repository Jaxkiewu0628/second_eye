## 核心定位（北极星）
- **产品**：帮助视障人士独立出行的 AI 眼镜
- **核心功能**：YOLO11n 实时识别障碍物 → 骨传导语音"前方 1.5 米台阶"
- **硬约束**：完全离线可用（对视障用户是安全要求，非可选特性）
- **形态**：眼镜框（轻量，~30g）+ 颈挂计算盒（V1 短线缆，V2 无线）

## 架构图（分体式）
```
眼镜框 (~30g)                  颈挂计算盒 (~150g)
┌──────────────────┐  ~25cm   ┌────────────────────┐
│ 摄像头 (MIPI)    │ ──线缆── │ RK3576 / RV1126B   │
│ VL53L5CX ToF     │          │ 5000mAh 电池        │
│ 骨传导振子        │          │ WiFi6 + BT5.3       │
│ 双麦克风          │          │ 颈挂/腰挂两用        │
│ ESP32-S3         │          └────────────────────┘
└──────────────────┘
端到端延迟：RK3576 ~185ms / RV1126B ~285ms（均 < 300ms 目标）
续航：5000mAh @ 1.1W = 13.7 小时（满足一天）
```

## 功能优先级

**V1（板到货即验证）**
- 障碍物检测 + 骨传导告警（YOLO11n + ToF 融合）
- 本地语音指令（STT → 规则推理 → TTS）
- 唤醒词（ESP32-S3 + WakeNet）
- 文字识别 OCR（按需触发）

**V1.5**
- 交通灯识别（红/绿）
- 斑马线检测
- 云端 LLM 场景描述

**V2**
- 完全无线（本地 WiFi P2P）
- 跌倒检测 + 紧急呼叫
- 单目深度估计（Depth Anything V2）

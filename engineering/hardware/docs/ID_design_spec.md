## 0. 文件目的

本文件定义外观/结构设计**必须遵守的内部预留腔体、开孔位置、装配通道与佩戴约束**。

> **核心承诺:** 只要设计满足本文所有"净空(clear)尺寸"为**硬下限、不允许负公差**,内部电子模块即可顺利装入。所有净空尺寸已包含装配间隙余量。设计可在净空之外自由发挥外观曲面。

本文不涉及内部电子方案细节,所有电子部件以**功能模块**抽象描述(如"主控电子模块""电池仓")。

## 1. 整机规格目标

| 指标 | 目标值 | 性质 |
|------|--------|------|
| 整机重量 | **< 50g** | 硬指标 |
| 防护等级 | **IPX4**(防汗水飞溅) | 硬指标 |
| 前框宽度 | 138–142mm | 参考 |
| 镜腿长度 | 143–147mm | 参考 |
| 镜腿最大厚度(横向 T) | **≤ 6mm** | 硬上限 |
| 镜腿最大高度(竖向 H) | ≤ 28mm | 参考 |
| 外壳表面温度 | **≤ 40°C**(皮肤接触安全) | 硬指标 |
| 佩戴方式 | 常规眼镜(鼻托 + 耳钩),可选运动绑带 | — |

---

## 2. 坐标系与命名(全文统一)

```
   X = 左右(宽度方向)
   Y = 前后(深度方向,镜片在前 −Y,镜腿向后 +Y)
   Z = 上下(高度方向)
```

镜腿三个方向术语:
- **长度 L** = 沿镜腿前后(铰链→耳),≈145mm
- **高度 H** = 竖向(刀锋的"宽脸")
- **厚度 T** = 横向(贴头←→外侧,刀锋的"薄边")

## 3. 整机分区与佩戴姿态

```
                   前框(包裹式)
        ┌────────────────────────────────┐
        │  摄像头   鼻托   [显示窗口]      │
        └──┬──────────────────────────┬──┘
         铰链L                       铰链R
      ┌───┴────────────┐      ┌───────┴──────────┐
      │  左镜腿(刀锋)  │      │  右镜腿(刀锋)    │
      │  电池 + 扬声器  │      │  主控模块仓+电池  │
      └────────────────┘      └──────────────────┘
   佩戴:前框贴脸,鼻托承重,两镜腿夹持并搭耳,耳钩段防滑。
```

**重量平衡原则:** 主控模块在右镜腿(偏重),左镜腿多配一节电池,左右重量差 **≤ 2g**。

## 4. 内部预留腔体清单(核心 · 净空为硬下限)

> "净空(clear)" = 腔体内部必须保证的最小净空间,**已含装配间隙**。设计的壁厚、曲面、加强筋必须在净空之外。

| 编号 | 腔体 | 净空尺寸 L×H×T (mm) | 位置 | 备注 |
|:---:|------|:---:|------|------|
| **C1** | **主控电子模块仓** | **30 × 24 × 5** | 右镜腿,铰链后 15–45mm 段 | 最大单体刚性件,决定该段外廓;外侧壁需散热(见 §7) |
| C2 | 电池仓(右) | 32 × 20 × 4 | 右镜腿,主控仓之后 | 软包电池,四周留 0.5mm 膨胀 |
| C3 | 电池仓(左 ×2) | 37 × 22 × 4(每个) | 左镜腿 | 同上,禁止挤压/折弯 |
| C4 | 摄像头仓 | 16 × 16 × 10 | 前框中央偏右 | 镜头前向开孔(见 O1) |
| C5 | 显示模块仓 | 25 × 15 × 8(暂定) | 右镜片后方 | ⚠️ 待选型确认,先按此预留 |
| C6 | 扬声器仓 ×2 | 22 × 11 × 4(每个) | 左右镜腿靠耳端 | 朝耳出声(见 O4) |
| C7 | 麦克风贴装位 ×2 | 3 × 3 × 2(每个) | 前框两端 | 配开孔 O3 |
| C8 | 副电子小板(左) | 18 × 10 × 2 | 左镜腿 | 音频/触控 |

> **C1 不可缩小、不可改形**:主控模块为一块刚性整体,30×24mm 是其平铺最小占地,5mm 是其叠层厚度。任何方向负公差都将导致无法装入。该段镜腿因此最厚最高(刀锋造型由此而来)。

## 5. 镜腿剖面(变截面 · 刀锋造型)

镜腿**不是等截面**。计算段最厚最高,向耳端渐变收窄,形成运动墨镜刀锋形态。

### 右镜腿(含主控模块)

```
段位        铰链段     主控刀锋段      电池过渡段     耳钩末端段
─────────────────────────────────────────────────────────────
长度 L      15mm       45mm           40mm          45mm
高度 H      20mm       26mm ──渐变──→  18mm ──渐变──→ 12mm
厚度 T      5mm        6mm            5mm           5mm
对应腔体    FPC通道    C1             C2            C6
```

### 左镜腿(外形对称)

```
段位        铰链段     电池刀锋段      过渡段        耳钩末端段
─────────────────────────────────────────────────────────────
高度 H      20mm       26mm ──渐变──→  18mm ──渐变──→ 12mm
厚度 T      5mm        5mm            5mm           5mm
对应腔体    FPC通道    C3             C8            C6
```

> 左右刀锋外形对称(美学统一);左侧无主控仓,可少 1mm 厚。

### 速记剖面(可直接示意)

```
右镜腿(俯视看厚度):  [铰链5] → [主控刀锋6] → [电池5] → [末端5]
右镜腿(侧视看高度):  [铰链20] → [主控刀锋26] → [渐变18] → [耳钩12]
左镜腿(侧视):        [铰链20] → [电池刀锋26] → [渐变18] → [耳钩12]
```

| 编号 | 开孔/凹槽 | 尺寸 | 位置 | 要求 |
|:---:|------|:---:|------|------|
| O1 | 摄像头镜头孔 | Φ8–10mm 圆孔 | 前框中央偏右 | 防眩光处理,镜头凸出 ≤3mm |
| O2 | 显示窗口 | 待定 | 右镜片区 | ⚠️ 待显示选型确认 |

## 7. 散热要求(主控模块仓 C1)

主控模块工作时发热,需被动散热,设计须满足:

1. **C1 外侧壁必须为金属**(铝合金),不可塑料。
2. 该金属壁**内表面平整**,预留 **≥ 26 × 45mm 平整接触面**(贴导热垫),此区域**禁止加强筋/凸台**。
3. 散热金属面**朝外侧(迎风)**,**不可朝皮肤面**——避免烫太阳穴,并借运动气流散热。
4. 该段可与镜腿其余部分用不同材质拼接(金属散热段 + 塑料减重段)。

---

## 8. 装配与走线通道

| 项 | 要求 |
|----|------|
| **前框↔镜腿柔性排线(FPC)通道** | 铰链处预留 **≥ 4 × 4mm** 弯折净空,贯穿铰链;弯折半径 **≥ 2mm**,弯折寿命 **> 50,000 次** |
| 铰链机械强度 | 既要转动开合,又要保护内部排线,刀锋薄壁处需加强 |
| 壳壁厚 | 塑料 **≥ 1.0mm**,金属 **≥ 0.8mm** |
| 装配间隙 | 各腔体壁与内件间 **≥ 0.3mm**(已含在净空内) |
| 开合/分件 | 镜腿需可开壳维修/装配,建议卡扣 + 螺丝复合 |

---

## 9. 佩戴与人体工学(运动场景)

1. **鼻托**:可调硅胶,承主要重量,运动防滑。
2. **耳钩段**:末端 40mm 硅胶包覆,防挥杆/跑动滑脱。
3. **重心**:重模块靠近铰链(耳支撑点),减小鼻梁压力与晃动力臂。
4. **帽子兼容**:铰链段镜腿内侧需避让帽带/帽檐(用户常戴运动帽)——请用真实运动帽验证。
5. **左右平衡**:外形对称,重量差 ≤ 2g。
6. 可选:预留运动绑带挂点(耳钩末端)。

---

## 10. 材料建议

| 部位 | 建议材质 | 理由 |
|------|---------|------|
| 主控刀锋段(C1 外壁) | 铝合金 | 散热(强制) |
| 镜腿其余段 | 塑料(PC/尼龙)或同铝合金 | 减重 / 质感统一(二选一) |
| 前框 | 塑料 + 包裹镜片 | 轻量 |
| 鼻托 / 耳钩 | 硅胶 | 防滑舒适 |

## 11. 待 ID / 结构确认事项

1. 🔴 **刀锋高度 26mm 是否在运动墨镜审美范围**——计算段必须 ≥24mm 净高,否则装不下。
2. 🔴 **帽子兼容**——铰链段弧度避让,需实物验证。
3. 🟡 **显示窗口 O2 / 显示仓 C5**——待显示模块选型,先按暂定值预留。
4. 🟡 **金属段与塑料段拼接工艺**——衔接强度与外观。
5. 🟡 **包裹单片镜与右眼显示区结合方式**——开窗 vs 内嵌。
6. 🟢 **左右对称 vs 左侧减薄**——是否为减重允许左侧略薄。

---

## 12. 验收准则

设计稿满足以下全部即视为"可装配通过":

- [ ] 所有腔体(C1–C8)净空尺寸达标,无负公差,内件 + 0.3mm 间隙可入
- [ ] C1 外侧金属平整散热面 ≥ 26×45mm,无加强筋,朝外
- [ ] FPC 通道弯折半径 ≥ 2mm,净空 ≥ 4×4mm
- [ ] 所有开孔/凹槽(O1–O6)位置与尺寸符合
- [ ] 整机重量 < 50g,左右差 ≤ 2g
- [ ] 防护达 IPX4,表面温度 ≤ 40°C
- [ ] 镜腿最大厚度 ≤ 6mm

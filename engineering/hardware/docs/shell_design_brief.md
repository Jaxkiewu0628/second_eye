# AR 运动眼镜外壳设计尺寸简报

> 供外观/工业设计师参考 · 2026-06-26  
> 目标：工程样机阶段，外壳需容纳所有内部器件并留合理走线空间

---

## 一、设计目标与约束

| 指标 | 目标 | 说明 |
|------|------|------|
| 整机重量 | **< 50g** | 行业主流 48-51g（Ray-Ban 48g，闪极 50g） |
| 佩戴感 | 接近普通眼镜 | 镜框不能太厚，鼻托可调 |
| 防护等级 | IPX4（运动） | 防汗水飞溅，接缝需设计排水槽 |
| 散热方式 | 被动散热 | 镜腿铝合金导热，无风扇 |
| 充电接口位置 | 镜腿末端 | 磁吸 Pogo Pin 或 USB-C |

---

## 二、整机结构分区

```
         ┌──────────────────────────────────────────┐
         │              前框（Front Frame）           │
         │  摄像头  ·  麦克风  ·  显示模块  ·  鼻托   │
         └──────┬───────────────────────────┬────────┘
                │ 左铰链（FPC 过线）          │ 右铰链（FPC 过线）
         ┌──────┴──────────┐     ┌──────────┴──────────┐
         │   左镜腿         │     │   右镜腿              │
         │  电池 × 2        │     │  主板 + 电池 × 1     │
         │  扬声器           │     │  扬声器  ·  充电口    │
         └─────────────────┘     └────────────────────┘
```

> **重量平衡原则：** 主板在右侧（较重），左侧电池数量多一节，使左右镜腿重量尽量相当。

### 3.2 右镜腿（主计算）— 尺寸现实校正

> ⚠️ **先纠正一个根本误区：35mm 的板子、甚至 RK3576 芯片本身，都塞不进等宽 14mm 的镜腿。**

#### 为什么 14mm 等宽镜腿对"计算区"不成立

镜腿横截面的 14mm 指的是**宽度**。PCB 平铺时，长边可以沿镜腿长度方向（镜腿长 145mm，够），但**短边（板宽）必须横着塞进镜腿宽度里**。

而 RK3576 芯片(FCBGA)本身就约 **15×15mm**，加上紧贴的 LPDDR5、走线和板边，**单是 SoC 那一块刚性板，最小也要 ~20×25mm 宽**。20mm 的宽度 > 14mm，**物理上放不进等宽 14mm 镜腿**。

> 一句话讲给外观组：**"主芯片这一块是硬骨头，它那段镜腿必须加粗到 ~20mm，或者把芯片挪到前框。其余部分才能做细。"**

#### 器件实际占板尺寸（硬数据）

| 器件 | 占板尺寸 | 高度 | 备注 |
|------|---------|------|------|
| **RK3576 (FCBGA)** | **~15 × 15mm** | ~1.2mm | **硬下限，决定板子最小宽度** |
| LPDDR5 内存 | ~10 × 12mm | ~1.0mm | 须距 SoC <15mm，通常紧贴 |
| eMMC 32GB 存储 | ~11 × 10mm | ~1.0mm | |
| PMIC 电源管理 | ~4 × 4mm | ~1mm | |
| 唤醒词芯片(NDP120) | ~3 × 3mm | ~1mm | 焊在主板，不单独占空间 |
| 去耦电容/电阻等 | 散布 SoC 周边 | — | |
| **→ 最小刚性"计算岛"** | **~20 × 25mm** | 双面贴装 ~3mm | **这块决定镜腿最粗处** |

#### 三种放置策略（三选一或组合）

| 方案 | 做法 | 镜腿外观 | 竞品参考 |
|------|------|---------|---------|
| **A. 铰链计算舱** | 铰链后做一段加粗"舱"，之后渐变收窄 | 铰链处可见加粗段 | Ray-Ban Meta(粗壮镜腿) |
| **B. 前框计算** | SoC 簇放前框眉心/桥区，FPC 走线到镜腿只放电池+音频 | 镜腿纤细，前框略厚 | 部分一体机 |
| **C. 刚柔分布** | SoC 一块不可避免的刚性"岛"，其余拆小板 FPC 串联分散 | 综合最优 | 商用眼镜通用做法 |

> 推荐 **A + C 组合**：计算岛放铰链舱，电池/音频/麦克风拆小板用 FPC 分散，整体平衡。

#### 修正后的右镜腿剖面（变截面，不是等宽）

```
            计算舱段          电池段           扬声器/末端段
长度        ~35mm            ~40mm            ~50mm
宽度        18-22mm   ──渐变──→  14mm   ──渐变──→  10-12mm
高度        10-11mm           8mm              8mm
内容        计算岛+散热        电池             扬声器+充电口
```

**右镜腿长度分配（共约 145mm）：**
```
铰链弯折区（FPC 过线，不放器件）：15mm
计算舱区（加粗到 18-22mm 宽）：    35mm
电池区（14mm 宽）：                40mm
扬声器区（收窄）：                 25mm
充电口 + 末端：                   30mm
```

**散热：** 计算岛背面贴 0.3mm 铜片，导热到计算舱段的铝合金外壳（这段加粗反而有利于散热面积）。

### 3.3 左镜腿（电池为主）

> 电池必须做成**窄长条**才能贴合 14mm 镜腿（18mm 宽的方块同样塞不进）。

| 器件 | 占板/占用尺寸 | 厚度 | 备注 |
|------|------|------|------|
| 电池（2节，各 ~100mAh） | **~50 × 10 × 4**（×2） | 4mm | 窄长条锂聚合物，沿镜腿排列 |
| 扬声器 | 9 × 20 | 3mm | 立体声，靠耳端 |
| 小 PCB（BT/音频） | ~10 × 18 | 0.8mm | 窄条，接扬声器 + 触控 |
| 触控传感器 | 贴镜腿表面 | — | 电容触摸，无开口 |

**左镜腿横截面需求：**
```
宽度：14mm（电池条 10mm 宽 + 走线 + 壳壁，留余量）
高度：8mm
```
> 左侧无计算岛，可全程保持 14mm 等宽，外观比右侧纤细——这也是为什么把重的计算放右侧、用左侧多一节电池配重。

---

### 3.4 铰链区（FPC 过线）

铰链是最脆弱的区域，FPC（柔性电路板）需要在此弯折通过。

| 参数 | 数值 | 说明 |
|------|------|------|
| FPC 厚度 | 0.1-0.18mm | 聚酰亚胺（PI）基材 |
| FPC 宽度 | 6-10mm | 包含电源 + 数据线 |
| 弯折半径 | ≥ 2mm | 过小会断裂 |
| 铰链内腔净空 | ≥ 4mm × 4mm | 供 FPC 弯折通过 |
| 弯折寿命 | > 50,000 次 | 延压铜 18µm |

## 五、散热设计要求

RK3576 在激活状态峰值功耗约 600mW，持续时间 3-10 秒/次（占空比工作模式）。

```
SoC 发热区（35×28mm PCB）
    │ 导热硅脂
    ▼
铜箔散热片（0.3mm）
    │ 导热胶带
    ▼
镜腿铝合金外壳（整体导热面积约 145×14mm）
    │ 自然对流 + 皮肤散热
    ▼
环境空气
```

**要求外壳设计师：**
- 右镜腿外壳优先使用铝合金（非塑料），导热面积越大越好
- SoC 区域对应的外壳内侧留平整接触面（不设加强筋），供贴导热片
- 外壳表面温度设计上限：**40°C**（皮肤接触安全）

## 七、需要外观设计师确认的问题

1. **镜腿截面形状**：方形截面 vs 椭圆截面？椭圆更舒适但内部空间不规则，影响 PCB 布局
2. **前框厚度**：摄像头模组需要 8-12mm 深度，前框视觉厚度可以接受吗？
3. **镜腿与头部接触区段**：靠耳段 40mm 是最重要的舒适区，这部分是否可以改用柔性材料（硅胶包覆）？
4. **充电口形式**：Pogo Pin 磁吸（需对齐）vs USB-C（需插拔，耐用性好）？
5. **重量分布**：右镜腿预计比左侧重 2-3g，需要通过材料/结构补偿，还是可以接受轻微不对称？
6. **显示模块开口**：等显示模块选型后，前框右侧镜片区需要为波导或 Micro-OLED 开口，尺寸待定
